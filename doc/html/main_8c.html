◆ decode_morse()

char decode_morse ( const char * code)

Wandelt einen Morsecode-String in ein einzelnes Zeichen zurück.

Die Funktion vergleicht den übergebenen Morsecode mit der Mapping-Tabelle und gibt das zugehörige Zeichen zurück. Bei einem unbekannten Code wird ein Stern ('*') zurückgegeben.

Parameters
codeNullterminierter String mit einem Morsecode (z. B. ".-" für A).
Returns
Der Klartextbuchstabe (Großbuchstabe, Ziffer oder Symbol) oder '*' bei Fehler.

◆ decode_text()

void decode_text ( const char * morse_input)

Dekodiert einen Morsecode-Text zu Klartext und gibt ihn aus.

Wandelt Morsecode zurück in Klartext und gibt das Ergebnis aus.

Die Funktion verarbeitet Morsezeichen, die durch Leerzeichen oder Slashes getrennt sind. Einfache Leerzeichen trennen Buchstaben, drei Leerzeichen oder ein Slash trennen Wörter.

Ungültige Morsezeichen werden durch '*' ersetzt. Zeilenumbrüche werden ignoriert.

Parameters
morse_inputNullterminierter String mit Morsecode.

◆ encode_char()

const char * encode_char ( char c)

Konvertiert ein einzelnes Zeichen in den zugehörigen Morsecode.

Konvertiert ein einzelnes alphanumerisches Zeichen in Morsecode.

Das übergebene Zeichen wird in Großbuchstaben umgewandelt und mit der internen Morsecode-Tabelle verglichen. Wird keine passende Zuordnung gefunden, gibt die Funktion einen Stern ('*') zurück.

Parameters
cDas zu konvertierende Zeichen.
Returns
Zeiger auf den Morsecode-String oder "*" bei unbekanntem Zeichen.

◆ encode_text()

void encode_text ( const char * text,
bool slash_mode )

Codiert einen vollständigen Text in Morsecode und gibt ihn auf stdout aus.

Wandelt einen kompletten Text in Morsecode um und gibt ihn aus.

Die Funktion wandelt Zeichen des Eingabetexts entsprechend der Mapping-Tabelle in Morsecode um. Nicht unterstützte Zeichen werden als Stern ('*') dargestellt.

Buchstaben werden durch ein Leerzeichen getrennt, Wörter wahlweise durch drei Leerzeichen oder (wenn slash_mode aktiviert ist) durch den Separator " / ".

Parameters
textNullterminierter C-String mit dem Eingabetext.
slash_modetrue = Worttrennung mit Slash; false = drei Leerzeichen.
