Detailed Description

Header-Datei für das Morsecode-Tool.

Diese Datei deklariert alle öffentlich zugänglichen Funktionen und Konstanten, die zum Codieren und Dekodieren von Morsezeichen benötigt werden. Sie wird von der main.c verwendet und trennt Header- und Implementierungscode.

char decode_morse ( const char * code)

Wandelt einen Morsecode-String in ein einzelnes Zeichen zurück.

Diese Funktion durchsucht die Morsecode-Mapping-Tabelle nach einer exakten Übereinstimmung mit dem übergebenen Morsecode-String und gibt das zugehörige Zeichen zurück. Bei ungültigen oder unbekannten Morsezeichen wird '*' zurückgegeben.

Parameters
codeMorsecode als nullterminierter String (z. B. "--..--").
Returns
Entsprechendes Klartextzeichen (A–Z, 0–9, Symbol) oder '*' bei unbekanntem Code.

Die Funktion vergleicht den übergebenen Morsecode mit der Mapping-Tabelle und gibt das zugehörige Zeichen zurück. Bei einem unbekannten Code wird ein Stern ('*') zurückgegeben.

Parameters
codeNullterminierter String mit einem Morsecode (z. B. ".-" für A).
Returns
Der Klartextbuchstabe (Großbuchstabe, Ziffer oder Symbol) oder '*' bei Fehler.

Wandelt Morsecode zurück in Klartext und gibt das Ergebnis aus.

Diese Funktion verarbeitet eine Morsecode-Zeile, bei der Morsezeichen durch Leerzeichen und Wörter durch drei Leerzeichen oder "/" getrennt sind. Unbekannte Morsezeichen werden durch '*' ersetzt.

Parameters
morse_inputNullterminierter String mit Morsecode.

Wandelt Morsecode zurück in Klartext und gibt das Ergebnis aus.

Die Funktion verarbeitet Morsezeichen, die durch Leerzeichen oder Slashes getrennt sind. Einfache Leerzeichen trennen Buchstaben, drei Leerzeichen oder ein Slash trennen Wörter.

Ungültige Morsezeichen werden durch '*' ersetzt. Zeilenumbrüche werden ignoriert.

const char * encode_char ( char c)

Konvertiert ein einzelnes alphanumerisches Zeichen in Morsecode.

Diese Funktion sucht das übergebene Zeichen in der internen Mapping-Tabelle und gibt den zugehörigen Morsecode-String zurück. Zeichen, die nicht enthalten sind (z. B. Umlaute oder Sonderzeichen außerhalb des definierten Umfangs), werden durch einen Stern ('*') dargestellt.

Parameters
cDas zu codierende Zeichen (Groß-/Kleinbuchstabe, Ziffer oder Symbol).
Returns
Zeiger auf den Morsecode als nullterminierten String oder "*" bei Fehler.

Konvertiert ein einzelnes alphanumerisches Zeichen in Morsecode.

Das übergebene Zeichen wird in Großbuchstaben umgewandelt und mit der internen Morsecode-Tabelle verglichen. Wird keine passende Zuordnung gefunden, gibt die Funktion einen Stern ('*') zurück.

Parameters
cDas zu konvertierende Zeichen.
Returns
Zeiger auf den Morsecode-String oder "*" bei unbekanntem Zeichen.

◆ encode_text()

void encode_text ( const char * text,
bool slash_mode )

Wandelt einen kompletten Text in Morsecode um und gibt ihn aus.

Die Funktion nimmt einen beliebigen C-String entgegen und konvertiert jedes Zeichen in das entsprechende Morsezeichen. Wörter werden je nach Einstellung durch " / " oder drei Leerzeichen getrennt. Nicht abbildbare Zeichen führen zu einem Stern ('*') in der Ausgabe.

Parameters
textEingabetext als nullterminierter C-String.
slash_modetrue, wenn Worttrennung mit "/" erfolgen soll; false für 3* "SPACE".

Wandelt einen kompletten Text in Morsecode um und gibt ihn aus.

Die Funktion wandelt Zeichen des Eingabetexts entsprechend der Mapping-Tabelle in Morsecode um. Nicht unterstützte Zeichen werden als Stern ('*') dargestellt.

Buchstaben werden durch ein Leerzeichen getrennt, Wörter wahlweise durch drei Leerzeichen oder (wenn slash_mode aktiviert ist) durch den Separator " / ".

Parameters
textNullterminierter C-String mit dem Eingabetext.
slash_modetrue = Worttrennung mit Slash; false = drei Leerzeichen.
